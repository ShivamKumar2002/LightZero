Welcome to LightZero's Documentation
=====================================

Overview
-------------

`LightZero <https://arxiv.org/pdf/2310.08348.pdf>`_ is a lightweight, efficient, and easy-to-understand `open-source algorithm toolkit <https://github.com/opendilab/LightZero>`_ that combines Monte Carlo Tree Search (MCTS) and Deep Reinforcement Learning (RL).
